# 简介

这是一个简易的记账本程序，现支持查看账单、新增账单等功能。

# 技术栈

vue + vuex + vue-cli + ES6/7 + axios + node + less + element-ui

# 项目运行

```	
# 克隆项目
(git bash) git clone https://github.com/SandraMeng/PocketBook

# 安装依赖
cd pocket-book 
npm install 或 yarn

cd admin 
npm install 或 yarn

# 启动服务
cd admin
node server.js

cd pocket-book
npm run serve 或 yarn serve

# 访问项目
访问 http://localhost:8080
```
